 **Learning JavaScript: Strings & Date Basics**

Today I practiced some useful JavaScript concepts.

 **Strings**

```javascript
let name = "bunny";

// Convert to uppercase
console.log(name.toUpperCase());

// Check length of string
console.log(name.length);
```

🔹 **Checking and Replacing Text**

```javascript
let text = "I LOVE BUNNY";

// Check if "BUNNY" exists in the text
console.log(text.includes("BUNNY"));

// Replace "BUNNY" with "MILI"
console.log(text.replace("BUNNY", "MILI"));
```

🔹 **Working with Dates**

```javascript
let now = new Date();

// Current date and time
console.log(now);

// Current hour
console.log(now.getHours());

// Current second
console.log(now.getSeconds());

// Current date (day of the month)
console.log(now.getDate());
```

✨ These simple methods are very useful when working with **text processing and timestamps in JavaScript applications.**
